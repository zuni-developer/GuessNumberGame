# Guess The Number Game

A simple and fun **number guessing game** built with **HTML, CSS, and JavaScript**.
The player tries to guess a secret number between **1 and 20** with limited attempts and gets feedback after each guess.

---

## Overview

This project is a classic “Guess the Number” game where the system randomly selects a number, and the user has to guess it.
The game provides hints like **“Too High”** or **“Too Low”**, and keeps track of score and highscore.

It mainly focuses on **DOM manipulation, game logic, and interactive UI updates using JavaScript**.

---

## Features

* Random number generation (1–20)
* Live feedback messages (Too High / Too Low / Correct)
* Score system (starts from 20 and decreases on wrong guesses)
* Highscore tracking
* Reset game with “Again!” button
* Dynamic UI changes (background color + number reveal)

---

## How to Play

1. Enter a number between **1 and 20**
2. Click **Check!**
3. Follow the hints:

   * Too High
   * Too Low
   * Correct Guess
4. Try to get the highest score before you lose
5. Click **Again!** to restart the game
